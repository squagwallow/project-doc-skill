# BF-01 — Metricool → Notion Social Reporting

| Field | Value |
|---|---|
| Current flow | Jo posts via Metricool; Priya manually screenshots + pastes weekly |
| Proposed flow | Metricool → Zapier weekly trigger → Notion rows; chart embedding subcontracted |
| Current owner | Priya |
| Proposed owner | Jo |
| Hours saved / week | 3.5 |
| Priority | P1 |
| Status | Proposed |

## Rationale

Highest time recovery for Priya (3.5h/wk). Jo knows Metricool better than Priya already. Automation eliminates manual transcription; Jo becomes process owner by default because she owns the upstream workflow.

## Dependencies

- Zapier plan sufficient for Metricool trigger + Notion action
- Chart embedding: subcontracted (see [metricool-chart-embed](../subcontractor-scopes/metricool-chart-embed))
